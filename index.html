プライバシーポリシー

「LiftLink AI」（以下「本アプリ」）は、ユーザーのトレーニング体験を向上させるため、以下の通りプライバシーポリシーを定めます。

1. 取得する情報

本アプリでは、以下の情報を取得および利用します。

  • 身体情報: 身長、体重、性別、生年月日。これらは適切なトレーニング強度の算出（1RM計算など）に使用します。
  • トレーニング記録: 種目名、重量、回数、セット数、実施日時。これらは進捗のグラフ表示や履歴管理に使用します。
  • ユーザー識別子: 表示名、および認証用の識別子（メールアドレス等）。これらはデータの同期とフレンド機能に使用します。
  • AI対話データ: AIトレーナーへ送信された質問内容。これらはアドバイスの生成に使用します。

2. 第三者提供と外部ツール

本アプリは、機能提供のために以下の外部サービスを利用します。送信されるデータは、それぞれのサービスのプライバシーポリシーに従って管理されます。

  • Google Gemini API (Google LLC): トレーニングのアドバイス生成のため、入力されたテキストおよび身体情報の要約を送信します。
  • Supabase (Supabase, Inc.): ユーザー認証およびデータのセキュアな保存のために利用します。

3. データの保存と削除

ユーザーデータは、Supabaseのクラウドストレージおよびデバイス上のセキュアな領域に保存されます。ユーザーがアカウントを削除した場合、または削除依頼があった場合、速やかにサーバー上の関連データを消去します。

4. 免責事項

本アプリが提供するAIのアドバイスや計算結果は参考情報であり、医学的な助言や安全性を保証するものではありません。トレーニングは自身の体調に合わせて無理のない範囲で行ってください。
